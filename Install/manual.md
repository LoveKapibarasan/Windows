## To disable widgets in Windows, follow these steps:
* Using Taskbar Settings:
1. Right-click on the taskbar.
2. Select “Taskbar settings.”
3. Toggle off the “Widgets” option.

## Steps to Disable News Feed on Microsoft Edge
1. Open Microsoft Edge.
2. Open a New Tab (`Ctrl`+`T`).
3. On the top-right corner of the New Tab page, click the gear icon (⚙️) for Page settings.
