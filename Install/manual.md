## 1. To disable widgets in Window:
* Using Taskbar Settings:
1. Right-click on the taskbar.
2. Select “Taskbar settings.”
3. Toggle off the “Widgets” option.

## 2. Disable News Feed on Microsoft Edge
1. Open Microsoft Edge.
2. Open a New Tab (`Ctrl`+`T`).
3. On the top-right corner of the New Tab page, click the gear icon (⚙️) for Page settings.


## Background
1. `Right-click` on an empty area of the desktop.
    * Select Personalize.
    * This opens `Settings` → `Personalization` → `Background`.

2. Under Background, choose the type:
    * Picture: Use a single image.
    * Solid color: Choose a simple color.
    * Slideshow: Cycle through multiple images automatically.


## 4. Uninstall
* Go to `Settings` → `Apps` → `Installed apps`
* Remove the app.
  * News
  * Weather
  * Solitaire
  * Paint
  * XBox
  * Calcurator
  * Microsoft Store
